1. FarmEasy (Farmers Management)

Overview:

FarmEasy is a specialized e-commerce platform that enables farmers to buy and sell saplings directly to one another, creating a peer-to-peer marketplace. The platform supports secure transactions and enhances accessibility for agricultural resources.


Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: Node.js, SQL (MySQL), XAMPP Server
Database: MySQL for storing user data, sapling listings, transactions.
2. Expensy (Expense Tracker Application)

Overview:

Expensy is a personal finance application that helps users keep track of their expenses, categorize spending, and analyze financial habits. It provides real-time insights into daily, weekly, or monthly expenditures.


Tech Stack:
Frontend: ReactJS, JavaScript, HTML
Backend: Node.js, SQL (MySQL), XAMPP Server
Database: SQL (for storing expenses and categories)
3. Traffic Sign Recognition and Voice Generation

Overview:

This project focuses on building an intelligent traffic sign recognition system using machine learning (Support Vector Machine). It enhances driver assistance by recognizing traffic signs generating corresponding audio alerts


Tech Stack:
Frontend: ReactJS, JavaScript, HTML
Languages: Python
Libraries: Node.js, SQL (MySQL), XAMPP Server
Use Cases: Driver assistance systems, autonomous vehicles, road safety systems.
